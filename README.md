## Moswb （Make Open Source World Better）

共产经济体的目标在于推进社会共产化并消灭货币。

简单说就是让当前社会的金钱游戏变为一场真正的游戏，

在共产社会，赚钱对于一个人就只是在玩一场游戏，

钱只是你在人生这场游戏里获得的分数。
